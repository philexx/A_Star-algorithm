# A_Start-algorithm


Press 'S' to pick the starting point.

Press 'D' to pick the destination point.

Press 'W' to place walls ( not walkable nodes ) 

Press 'ENTER' to "find/create the path"

Press 'C' to delete a singe node.

Press 'DEL' to clear
